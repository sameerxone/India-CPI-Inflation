# India CPI Inflation Analysis

## 📌 Project Overview

This project analyzes **India's Consumer Price Index (CPI) inflation** to understand inflation trends and the major factors influencing price changes. The analysis focuses on identifying patterns in CPI data and examining the contribution of components such as **food, fuel, and core inflation**.

## 🎯 Objectives

* Analyze historical CPI data for India.
* Identify **inflation trends over time**.
* Understand the impact of different CPI components on overall inflation.
* Visualize inflation patterns to make insights easier to interpret.

## 📊 Dataset

The dataset contains **CPI inflation data for India** collected from publicly available economic sources.
It includes indicators such as:

* Overall CPI inflation
* Food inflation
* Fuel inflation
* Core inflation
* Monthly / yearly inflation changes

## 🛠 Tools & Technologies

* **Microsoft Excel**
* Data Cleaning
* Data Visualization
* Trend Analysis

## 📈 Analysis Performed

* Data cleaning and preprocessing
* Trend analysis of CPI inflation
* Component-wise inflation comparison
* Visualization of inflation patterns using charts

## 🔍 Key Insights

* Food prices often contribute significantly to CPI inflation fluctuations.
* Fuel price changes can create short-term spikes in inflation.
* Core inflation helps identify underlying long-term price trends.

## 📷 Sample Visualizations

Charts and graphs were created to illustrate:

* Monthly inflation trends
* Year-on-year CPI changes
* Contribution of different CPI components
